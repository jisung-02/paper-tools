    How to edit PDF metadata

    1. Drop a PDF file into the dropzone.
    2. Edit the Title, Author, Subject or Keywords fields (leave a field blank to keep its existing value), or check "Remove all metadata".
    3. Click "Save" to download the updated PDF.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your file never leaves your device.

    What happens if I leave a field blank?

    The existing value for that field is kept unchanged, unless "Remove all metadata" is checked, which clears everything regardless of the fields.
