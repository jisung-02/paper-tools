      How to build a PDF workflow

      1. Drop in one or more PDFs, then add the steps you want to run, in order.
      2. Reorder, edit or remove steps, and adjust each step's parameters as JSON.
      3. Click Run workflow to chain every step, then download the final PDF.

      FAQ

      Do I have to re-upload the file between steps?

      No — the output of each step is fed directly into the next one, and only the final result is downloaded.

      Can I save a workflow and reuse it later?

      Yes — export your steps as a JSON file, then import it later to rebuild the same workflow.

      Are my PDFs uploaded to a server?

      No — every step runs locally in your browser, and your files never leave your device.
