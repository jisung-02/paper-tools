    How to reorder PDF pages

    1. Drop a PDF file into the dropzone.
    2. Enter the new order, listing every page number once, e.g. "3,1,2".
    3. Click "Reorder" to download the PDF in the new page order.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your file never leaves your device.

    Do I need to list every page number?

    Yes — the new order must include every page number in the document exactly once; incomplete or duplicated lists are rejected.
