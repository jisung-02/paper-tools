    How to password-protect a PDF

    1. Drop a PDF file into the dropzone.
    2. Enter a user password (required to open the file) and, optionally, an owner password (leave blank to match the user password).
    3. Click "Set Password" to download the encrypted PDF.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your file never leaves your device.

    How strong is the encryption?

    You can choose AES-256 (recommended) or AES-128, using Latin letters and numbers for the password.
