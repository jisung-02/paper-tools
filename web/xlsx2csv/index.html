    How to convert Excel to CSV

    1. Drop an Excel (.xlsx) file into the dropzone.
    2. Wait for the tool to finish loading.
    3. Click "Convert" to download a .csv file, or a .zip of .csv files if the workbook has multiple sheets.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser; your file never leaves your device.

    What happens if my workbook has multiple sheets?

    Each sheet becomes its own .csv file, packaged together in a single .zip download. A workbook with only one sheet downloads as a plain .csv file instead.
