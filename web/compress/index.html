    How to compress a PDF

    1. Drop a PDF file into the dropzone.
    2. Choose a compression preset, or adjust quality, target image size and grayscale manually.
    3. Click "Compress" to download the smaller PDF.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser; your file never leaves your device.

    How much smaller will my file get?

    It depends on the file — quality and target image size affect image-heavy PDFs most. Grayscale can shrink color scans, while text-only PDFs barely change.
