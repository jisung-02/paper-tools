    How to compress a PDF

    1. Drop a PDF file into the dropzone.
    2. Adjust the image quality slider and the max image width.
    3. Click "Compress" to download the smaller PDF.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your file never leaves your device.

    How much smaller will my file get?

    It depends on the file — Compress recompresses embedded images using lossy JPEG, so image-heavy PDFs shrink the most, while text-only PDFs barely change.
