    How to use the HTML editor

    1. Type or paste HTML in the editor, or drop a .html, .htm or .txt file onto it.
    2. Watch the rendered page update live as you type.
    3. Click "Download .html" to save your document.

    FAQ

    Is my HTML uploaded to a server?

    No — everything runs locally in your browser; your HTML never leaves your device.

    Do scripts in my HTML run in the preview?

    Yes — inside a sandboxed preview frame that is isolated from the rest of the page and your other files.

    Can I use it as a viewer for existing HTML files?

    Yes — drop an .html file onto the editor to view it; nothing is uploaded.
