      How to batch-process PDFs

      1. Choose an operation and drop in as many PDFs (or images) as you need.
      2. Pick Independent to run the operation on each file separately, or Grouped to combine every file into one output.
      3. Click Run batch, then download the results as one ZIP file.

      FAQ

      What happens if one file fails?

      A failed file is skipped and listed separately — it never stops the rest of the batch from finishing.

      What's the difference between Independent and Grouped mode?

      Independent applies the operation to each file on its own, producing one output per input. Grouped combines all files into a single output, for operations like merging.

      Are my files uploaded anywhere?

      No — every file is processed locally in your browser, and the ZIP is built and downloaded without ever leaving your device.
