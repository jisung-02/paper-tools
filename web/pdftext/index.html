    How to extract text from a PDF

    1. Drop a PDF file into the dropzone.
    2. Click "Extract Text".
    3. A .txt file with the selectable text downloads automatically.

    FAQ

    Is my file uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your file never leaves your device.

    Does this work on scanned PDFs?

    No — it extracts selectable text only; scanned or image-only PDFs have no embedded text to extract, since there is no OCR.
