    How to use the Markdown editor

    1. Type or paste Markdown in the editor, or drop a .md, .markdown or .txt file onto it.
    2. Watch the live preview update as you type.
    3. Click "Download .md" or "Download HTML" to save your document.

    FAQ

    Is my Markdown uploaded to a server?

    No — everything runs locally in your browser; your Markdown never leaves your device.

    What's in the downloaded HTML file?

    A standalone HTML file with your rendered Markdown and minimal built-in styling — no external files, fonts or scripts needed.
