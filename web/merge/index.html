    How to merge PDFs

    1. Drop two or more PDF files into the dropzone — they'll be combined in the order you add them.
    2. Check the file list to confirm the order is correct.
    3. Click "Merge" to download the combined PDF.

    FAQ

    Are my files uploaded to a server?

    No — everything runs locally in your browser via WebAssembly; your files never leave your device.

    Can I change the order of the files after adding them?

    Not directly — remove the files and re-add them in the order you want, since files are merged in the order you select them.
