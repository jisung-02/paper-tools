      FAQ

      Are my files uploaded to a server?

      No. The file moves directly between the two devices in your browser and is never uploaded to any server.

      Why do both devices need to be on the same network?

      This tool doesn't use a server to help the two devices find each other, so it only works when they can reach each other directly — usually true when both are on the same Wi-Fi or local network.

      What if the connection fails?

      Make sure both devices are on the same network and try again.
